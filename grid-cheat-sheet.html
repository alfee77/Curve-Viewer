Grid Recap

  • CSS Grid is the most advanced layout type.
  • Grid does not replace Flexbox. Both layouts have their use cases.
  • The element with display: grid is called the grid container.
  • The direct descendants of a grid container are called grid items.
  • The grid-template-columns defines the tracks of a grid in the column direction. It lets you define the number of tracks and their sizes. This property is defined on the grid container.
  • The grid-template-rows property allows you to define tracks in the row direction. This property only works on a grid container.
  • We'll use the fr unit to define responsive track sizes.
  • The gap property sets the gutters between the grid tracks.
  • When auto is used in combination with the fr unit, auto is calculated before fr.
  • The minmax() function takes two parameters, min and max, and defines an acceptable range from the min value to the max value.
  • repeat(auto-fit, minmax(150px, 1fr)) responsively fits as many 150px columns as possible. The remaining space is distributed.
  • The grid-column shorthand grid-column: grid-column-start / grid-column-end; allows you to set the start and end values in one go (separated with a forward slash /).
  • The grid-row property is used on a grid item and can be used to specify the start and end position on the row lines.
  • The grid-area accepts the following syntax: row-start / column-start / row-end / column-end.
  • grid-template-columns: [sidebar] 100px [main] 1fr [last]; defines two tracks with three (named) grid lines.
  • Implicit rows' sizes can be controlled by the grid-auto-rows property.
  • The default flow of grid items is row. It can be customized with the grid-auto-flow property.
  • grid-auto-flow: column would create new implicit columns if necessary.
  • Implicit columns' sizes can be controlled by the grid-auto-columns property.
  • The justify keyword refers to the inline direction.
  • The align keyword refers to the block direction.
  • The content keyword refers to the grid structure as a whole.
  • The items keyword refers to the grid items in relation to their grid cells.
  • display: grid; place-items: center is a quick way to center a grid layout with a single grid item.
